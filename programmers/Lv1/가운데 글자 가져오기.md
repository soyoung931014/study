----
https://school.programmers.co.kr/learn/courses/30/lessons/12903

## 풀이
```jsx
function solution(s) {
    let answer;
  if(s.length%2 === 0){
      let half = s.length/2
      answer = s.slice(half-1, half+1)
  }else{
      let half = parseInt(s.length/2)
      answer = s[half]
  }
    return answer
}
```

## 다른 풀이

function solution(s) {
    return s.substr(Math.ceil(s.length/2)-1, s.length%2 === 0 ? 2: 1)
}
