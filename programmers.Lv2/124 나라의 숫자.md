## 풀이
----

```jsx
 

function solution(n) {
  let arr = [4,1,2]
  let answer = ""
  while(n){
   answer = arr[n%3] + answer
   n = n%3 === 0 ? n/3-1 : Math.floor(n/3)    
}
    return answer

}
```

124를 배열로 만들면 [1,2,4] 0,1,2인덱스로 3진법으로 변환해서 풀이를 접근해보자.
