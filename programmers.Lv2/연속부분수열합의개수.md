```jsx
function solution(elements) {
  const set = new Set(); 
  const n = elements.length; 
  
  for(let i=0; i<n; i++){
    let sum = 0;
    for(let j=i; j<i+n; j++){
      sum += elements[j%n];
      set.add(sum)
    }

  }
  return set.size
}

원형 수열은 처음과 끝이 연결되어 끊기는 부분이 없어야 하기 때문에, elements를 순회할 수 있도록 만들어줘야한다.
문제에선 길이가 1인 연속 부분 수열로부터 [1, 4, 7, 9] 네 가지의 합이 나올 수 있습니다.

길이가 3인 연속 부분 수열로부터 [6, 11, 12, 17, 20] 다섯 가지의 합이 나올 수 있습니다.
```
